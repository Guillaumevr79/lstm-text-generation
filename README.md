# lstm-text-generation
Implémentation d'un modèle de langage basé sur les réseaux LSTM (Long Short-Term Memory) avec PyTorch. Ce projet explore la génération de texte à travers l'apprentissage des structures linguistiques. Inclut l'entraînement du modèle, différentes stratégies de décodage (greedy, sampling) et la génération de phrases.
